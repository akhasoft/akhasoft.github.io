Class Core.Utils.ToastBuilder

  • java.lang.Object
    • akha.yakhont.Core.Utils.ToastBuilder
  • Enclosing class:
    Core.Utils

    public static class Core.Utils.ToastBuilder
extends Object
    Builds and shows Toast with custom duration (please refer to setDuration(java.lang.Integer)).

    Can call Activity.onActivityResult(int, int, android.content.Intent) (see setRequestCode(java.lang.Integer)).
