Class CoreLogger

  • java.lang.Object
    • akha.yakhont.CoreLogger

  • public class CoreLogger
extends Object
    The CoreLogger class is responsible for logging. It also contains some additional components:

    • for recording audio / video: VideoRecorder
    • for making screenshots: getScreenshot(String)
    • for detecting (and handling) shaking device: ShakeEventListener
    • for gestures recognition (and handling too): GestureHandler

    And all these components could be used independently, i.e. not for logging only but for any purpose you need.

    Some of available logging features are:

    • No logging in release builds (except errors)
    • Stack trace support: log(Level, String, boolean)
    • Byte arrays logging: toHex(byte[]), toHex(byte[], int, int, Locale, Charset)
    • The logcat support: getLogCat(List, boolean, String)
    • Sending error reports via e-mail: sendLogCat(Activity, String...), sendData
    • Sending error reports via e-mail on shaking device (or making Z-gesture): registerDataSender(Context, String...)

    Call setLogLevel(akha.yakhont.CoreLogger.Level) to set log level.
    Call setShowStack(boolean) to force stack trace logging.
    Call setShowThread(boolean) to force thread info logging.
