Class CoreLogger.VideoRecorder

  • java.lang.Object
    • akha.yakhont.CoreLogger.VideoRecorder
  • Enclosing class:
    CoreLogger

    public static class CoreLogger.VideoRecorder
extends Object
    Component to record video / audio, requires API version >= LOLLIPOP.
    This is simplified version, intended for applications debug only; don't use it for professional audio / video recording.
    Usage example: 

     VideoRecorder.start(activity, outputFile);
 ...
 VideoRecorder.stop();
    Audio recording can be switched off by the dynamic permission Manifest.permission.RECORD_AUDIO - just don't provide it :-)
