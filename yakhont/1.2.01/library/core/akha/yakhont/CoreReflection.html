Class CoreReflection

  • java.lang.Object
    • akha.yakhont.CoreReflection

  • public class CoreReflection
extends Object
    The helper class for work with Java Reflection API.

    Some of available features are:

    • Working with arrays, collections, SparseArrays, etc.
      • Checks if parameter is array, collections, etc. isNotSingle(Object)
      • Gets the size of parameter (if it's an array, collections, etc.) getSize(Object)
      • Returns contained objects if parameter is array, collections, etc. getObjects(Object, boolean)
      • Merges data (each parameter can be array, collections, etc.) mergeObjects(Object, Object)
    • Finds list of the overridden methods findOverriddenMethods(Class, Class)
    • Compares methods equalsMethods(Method, Method)
    • Finds list of methods, based on selection flags findMethods
    • Returns object's fields getFields
