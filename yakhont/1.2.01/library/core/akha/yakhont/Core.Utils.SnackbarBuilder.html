Class Core.Utils.SnackbarBuilder

  • java.lang.Object
    • akha.yakhont.Core.Utils.SnackbarBuilder
  • Enclosing class:
    Core.Utils

    public static class Core.Utils.SnackbarBuilder
extends Object
    Builds and shows Snackbar with Views customization possibilities (please refer to setViewHandler(akha.yakhont.technology.Dagger2.UiModule.ViewModifier)).

    Sequential Snackbar's calls are queued.

    By default calls Activity.onActivityResult(int, int, android.content.Intent) (see setRequestCode(java.lang.Integer)).
