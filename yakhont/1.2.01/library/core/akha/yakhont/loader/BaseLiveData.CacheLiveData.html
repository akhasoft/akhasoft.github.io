    public static class BaseLiveData.CacheLiveData<D>
extends BaseLiveData<D>
    A class that performs loading of data. If network is not available, loading goes from cache (which is updated after every successful loading from network). Most implementations should not use CacheLiveData directly, but instead utilise Retrofit2LoaderWrapper or Retrofit2LoaderWrapper.Retrofit2LoaderBuilder.
