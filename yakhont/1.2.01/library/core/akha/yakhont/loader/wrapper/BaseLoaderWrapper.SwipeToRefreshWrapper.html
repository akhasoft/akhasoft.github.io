akha.yakhont.loader.wrapper

Class BaseLoaderWrapper.SwipeToRefreshWrapper

  • java.lang.Object
    • akha.yakhont.loader.wrapper.BaseLoaderWrapper.SwipeToRefreshWrapper
  • Enclosing class:
    BaseLoaderWrapper<D>

    public static class BaseLoaderWrapper.SwipeToRefreshWrapper
extends Object
    The SwipeToRefreshWrapper class is intended to support "swipe-to-refresh" feature (in case of PagedListAdapter the DataSource.invalidate() method will be called). To enable swipe-to-refresh, activity (or fragment) should contain SwipeRefreshLayout and register it via one of the methods provided. For example: 

     import akha.yakhont.loader.wrapper.BaseResponseLoaderWrapper.CoreLoad;

 public class YourFragment extends Fragment {

     @Override
     public void onActivityCreated(Bundle savedInstanceState) {
         super.onActivityCreated(savedInstanceState);

         CoreLoad coreLoad = ...

         SwipeToRefreshWrapper.register(R.id.swipeContainer, coreLoad);
     }
 }
    And in the layout XML: 

     <android.support.v4.widget.SwipeRefreshLayout
     android:id="@+id/swipeContainer"
     ... >

     <androidx.recyclerview.widget.RecyclerView ... />

 </android.support.v4.widget.SwipeRefreshLayout>
