akha.yakhont.loader.wrapper

Class BaseLoaderWrapper<D>

  • java.lang.Object
    • akha.yakhont.loader.wrapper.BaseLoaderWrapper<D>
  • Type Parameters:
    D - The type of data
    Direct Known Subclasses:
    BaseResponseLoaderWrapper

    public abstract class BaseLoaderWrapper<D>
extends Object
    The BaseLoaderWrapper class is a wrapper for data loading classes like BaseViewModel. It's designed to be a root class for standalone, independent, self-sufficient data loading components (hiding low-level implementation details and data loading classes's related boilerplate code). Most implementations should not use BaseLoaderWrapper directly, but instead utilise Retrofit2LoaderWrapper or Retrofit2LoaderWrapper.Retrofit2LoaderBuilder.
    See Also:
    BaseResponseLoaderWrapper, BaseViewModel
