      • setListItem

        @NonNull
public BaseResponseLoaderWrapper.CoreLoadBuilder<C,R,E,D> setListItem(@LayoutRes
                                                                     int layoutItemId)
        Sets the resource identifier of a layout file that defines the views to bind.

        If a layout ID was not set, the following algorithm will be applied:

        • find name of the list, which is the string representation of the list ID; if the ID was not defined, "list", "grid" or "recycler" will be used
        • look for the layout with ID equals name + "_item"; if not found, look for the layout with ID equals name
        • if not found and name ends with "View", removes that suffix; then goes to the previous step
        Parameters:
        layoutItemId - The resource identifier of a layout file that defines the views to bind
        Returns:
        This CoreLoadBuilder object to allow for chaining of calls to set methods
