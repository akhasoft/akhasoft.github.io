Class BaseResponseLoaderWrapper<C,R,E,D>

  • Type Parameters:
    C - The type of callback
    R - The type of network response
    E - The type of error (if any)
    D - The type of data to load
    Direct Known Subclasses:
    Retrofit2LoaderWrapper, RetrofitLoaderWrapper

    public abstract class BaseResponseLoaderWrapper<C,R,E,D>
extends BaseLoaderWrapper<BaseResponse<R,E,D>>
    Extends the BaseLoaderWrapper class to provide BaseResponse and cache support. Most implementations should not use BaseResponseLoaderWrapper directly, but instead utilise Retrofit2LoaderWrapper or Retrofit2LoaderWrapper.Retrofit2LoaderBuilder.
    See Also:
    BaseLiveData.CacheLiveData, BaseResponseLoaderWrapper.CoreLoad
