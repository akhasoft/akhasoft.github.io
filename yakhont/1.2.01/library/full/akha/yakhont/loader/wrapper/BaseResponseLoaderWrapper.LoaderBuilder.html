  • Type Parameters:

    public static interface BaseResponseLoaderWrapper.LoaderBuilder<C,R,E,D>
    The API to create new BaseResponseLoaderWrapper (or BaseLoaderWrapper) instances.

    First, createBaseResponseLoader() will be called. In case of returned null, the result of calling createBaseLoader() will be used.
