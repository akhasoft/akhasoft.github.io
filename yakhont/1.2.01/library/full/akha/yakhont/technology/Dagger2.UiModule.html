      • addToSnackbarsQueue

        public static boolean addToSnackbarsQueue(com.google.android.material.snackbar.Snackbar snackbar,
                          Integer requestCode,
                          Intent intent,
                          Activity activity,
                          boolean actionNoDefaultHandler,
                          CountDownLatch countDownLatch,
                          Long countDownLatchTimeout,
                          String id)
        Adds Snackbar to Snackbar's queue.
        Parameters:
        snackbar - The Snackbar
        requestCode - The request code to call Activity.onActivityResult(int, int, android.content.Intent) (or null for no call)
        intent - The intent to pass to Activity.onActivityResult(int, int, android.content.Intent) (or null)
        activity - The Activity (or null for default one)
        actionNoDefaultHandler - false to call Activity.onActivityResult(int, int, android.content.Intent) in case of Snackbar.Callback.DISMISS_EVENT_ACTION, true otherwise
        countDownLatch - The CountDownLatch (if any) to wait until run the next Snackbar from queue
        countDownLatchTimeout - The CountDownLatch timeout (or null)
        id - The Snackbar's ID (or null)
        Returns:
        true if added successfully, false otherwise
