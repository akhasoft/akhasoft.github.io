Class CoreLogger.ShakeEventListener

  • java.lang.Object
    • akha.yakhont.CoreLogger.ShakeEventListener
  • All Implemented Interfaces:
    SensorEventListener
    Enclosing class:
    CoreLogger

    public static class CoreLogger.ShakeEventListener
extends Object
implements SensorEventListener
    Simple shake device handler with customizable threshold. Usage example: 

     public class YourActivity extends Activity {

     @Override
     protected void onCreate(Bundle savedInstanceState) {
         super.onCreate(savedInstanceState);

         new ShakeEventListener(new Runnable() {
             @Override
             public void run() {
                 // your code here (to handle device shaking)
             }
         }).register();
     }
 }
