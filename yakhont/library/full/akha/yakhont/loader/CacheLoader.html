  • Type Parameters:
    C - The type of callback
    R - The type of network response
    E - The type of error (if any)
    D - The type of data in this loader

    public abstract class CacheLoader<C,R,E,D>
extends BaseLoader<C,R,E,D>
    A class that performs asynchronous loading of data. If network is not available, loading goes from cache (which is updated after every successful loading from network). Most implementations should not use CacheLoader directly, but instead utilise RetrofitLoaderWrapper or RetrofitLoaderWrapper.RetrofitLoaderBuilder.
