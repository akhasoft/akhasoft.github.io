Class BaseActivityLifecycleProceed.BaseActivityCallbacks

    public abstract static class BaseActivityLifecycleProceed.BaseActivityCallbacks
extends BaseCallbacks.BaseCacheCallbacks<Activity>
    Extends the BaseCallbacks.BaseCacheCallbacks class to provide the activity life cycle callbacks support. For the moment supported callbacks are similar to the Application.ActivityLifecycleCallbacks but will likely expand in future releases.
    By default all callbacks are empty.

    Usage example (for more examples please refer to general Activity, Fragment and simple Activity ones): 

     package com.mypackage;

 public class MyActivityCallbacks extends BaseCallbacks.BaseActivityCallbacks {

     @Override
     public void onActivityCreated(Activity activity, Bundle savedInstanceState) {

         // your code here (NOTE: you don't have to call activity.onActivityCreated() -
         //   it's already done by the Weaver)
     }
 }
    Annotate necessary Activities: 

     import akha.yakhont.callback.annotation.CallbacksInherited;

 @CallbacksInherited(com.mypackage.MyActivityCallbacks.class)
 public class MyActivity extends Activity {
     ...
 }
    And register your callbacks handler (see also Core): 

     import akha.yakhont.Core;

 import com.mypackage.MyActivityCallbacks;

 public class MyApplication extends Application {

     @Override
     public void onCreate() {
         super.onCreate();
         ...
         Core.run(this);

         BaseActivityLifecycleProceed.register(new MyActivityCallbacks());
         // OR
         // BaseActivityLifecycleProceed.register(
         //     (BaseActivityCallbacks) new MyActivityCallbacks().setForceProceed(true));
         // to apply callback handlers to ALL activities (annotated or not)
     }
 }
    Please refer to the BaseCallbacks for more details.
