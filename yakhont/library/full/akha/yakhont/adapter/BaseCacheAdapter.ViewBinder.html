    This class can be used by external clients of BaseCacheAdapter to bind values to views.

    • Method Detail

      • setViewValue

        boolean setViewValue(View view,
                   Object data,
                   String textRepresentation)
        Binds the value to the specified view. When binding is handled by this ViewBinder, this method must return VIEW_BOUND; otherwise, BaseCacheAdapter will attempts to handle the binding on its own.
        Parameters:
        view - The view to bind the data to
        data - The data to bind to the view
        textRepresentation - The String representation of the supplied data: it is either the result of data.toString() or an empty String but it is never null
        Returns:
        true if the value was bound to the view, false otherwise
