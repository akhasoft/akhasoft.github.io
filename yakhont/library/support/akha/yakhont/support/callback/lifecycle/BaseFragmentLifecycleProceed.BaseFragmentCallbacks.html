Class BaseFragmentLifecycleProceed.BaseFragmentCallbacks

    BaseFragmentLifecycleProceed

    public abstract static class BaseFragmentLifecycleProceed.BaseFragmentCallbacks
extends BaseCallbacks.BaseCacheCallbacks<Fragment>
    Extends the BaseCallbacks.BaseCacheCallbacks class to provide the fragment life cycle callbacks support. For the moment supported callbacks are close to the Application.ActivityLifecycleCallbacks but will likely expand in future releases.
    By default all callbacks are empty.

    Usage example (for more examples please refer to general Activity, Activity and simple Activity ones): 

     package com.mypackage;

 public class MyFragmentCallbacks extends BaseCallbacks.BaseFragmentCallbacks {

     @Override
     public void onFragmentCreated(Fragment fragment, Bundle savedInstanceState) {

         // your code here (NOTE: you don't have to call fragment.onFragmentCreated() -
         //   it's already done by the Weaver)
     }
 }
    Annotate necessary Fragments: 

     import akha.yakhont.callback.annotation.CallbacksInherited;

 @CallbacksInherited(com.mypackage.MyFragmentCallbacks.class)
 public class MyFragment extends Fragment {
     ...
 }
    And register your callbacks handler (see also Core): 

     import akha.yakhont.Core;

 import com.mypackage.MyFragmentCallbacks;

 public class MyApplication extends Application {

     @Override
     public void onCreate() {
         super.onCreate();
         ...
         Core.run(this);

         BaseFragmentLifecycleProceed.register(new MyFragmentCallbacks());
         // OR
         // BaseFragmentLifecycleProceed.register(
         //     (BaseFragmentCallbacks) new MyFragmentCallbacks().setForceProceed(true));
         // to apply callback handlers to ALL fragments (annotated or not)
     }
 }
    Please refer to the BaseCallbacks for more details.
