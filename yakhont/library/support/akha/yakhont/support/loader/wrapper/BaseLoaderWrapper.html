Class BaseLoaderWrapper<D>

  • java.lang.Object
    • akha.yakhont.support.loader.wrapper.BaseLoaderWrapper<D>

    D - The type of data
    All Implemented Interfaces:

    Direct Known Subclasses:
    BaseResponseLoaderWrapper

    public abstract class BaseLoaderWrapper<D>
extends Object
implements LoaderManager.LoaderCallbacks<D>
    The BaseLoaderWrapper class is a wrapper for LoaderManager and Loader instances associated with it. It's designed to be a root class for standalone, independent, self-sufficient data loading components (hiding low-level implementation details and LoaderManager's related boilerplate code). Most implementations should not use BaseLoaderWrapper directly, but instead utilise RetrofitLoaderWrapper or RetrofitLoaderWrapper.RetrofitLoaderBuilder.
