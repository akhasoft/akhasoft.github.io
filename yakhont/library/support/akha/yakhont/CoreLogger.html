Class CoreLogger

  • java.lang.Object
    • akha.yakhont.CoreLogger

  • public class CoreLogger
extends Object
    The CoreLogger class is responsible for logging. In addition to usability and flexible settings it provides such build-in features as auto-disable logging in release builds, stack-trace and thread info logging, support for 3-rd party loggers etc.

    There is also the possibility to send log records via e-mail (useful for hardly-reproduced errors).
