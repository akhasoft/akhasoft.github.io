Class CorePermissions

  • java.lang.Object
    • akha.yakhont.CorePermissions
  • All Implemented Interfaces:
    Core.ConfigurationChangedListener

    public class CorePermissions
extends Object
implements Core.ConfigurationChangedListener
    The helper class for work with Android Permissions API. Usage example: 

     import akha.yakhont.CorePermissions;

 import android.Manifest;

 new CorePermissions.RequestBuilder(activity, Manifest.permission.ACCESS_FINE_LOCATION)
     .setOnGranted(new Runnable() {
         @Override
         public void run() {
             // your code here, e.g. requestLocationUpdates()
         }
     })
     .request();
