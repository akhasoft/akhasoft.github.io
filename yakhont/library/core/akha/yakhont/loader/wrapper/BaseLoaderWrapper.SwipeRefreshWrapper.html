akha.yakhont.loader.wrapper

Class BaseLoaderWrapper.SwipeRefreshWrapper

  • java.lang.Object
    • akha.yakhont.loader.wrapper.BaseLoaderWrapper.SwipeRefreshWrapper
  • Enclosing class:
    BaseLoaderWrapper<D>

    public static class BaseLoaderWrapper.SwipeRefreshWrapper
extends Object
    The SwipeRefreshWrapper class is intended to support "swipe refresh" feature. To enable swipe refresh, fragment should contain SwipeRefreshLayout and register it via one of the methods provided (for example, register()). For example: 

     public class MyFragment extends Fragment {

     @Override
     public void onActivityCreated(Bundle savedInstanceState) {
         super.onActivityCreated(savedInstanceState);
         ...
         SwipeRefreshWrapper.register(this, R.id.swipeContainer);
     }
 }
    And in the layout XML: 

     <android.support.v4.widget.SwipeRefreshLayout
     android:id="@+id/swipeContainer"
     ... >

     <ListView ... />

 </android.support.v4.widget.SwipeRefreshLayout>
