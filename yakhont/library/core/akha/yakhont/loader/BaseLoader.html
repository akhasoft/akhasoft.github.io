    • android.content.Loader<BaseResponse<R,E,D>>

    C - The type of callback
    R - The type of network response
    E - The type of error (if any)
    D - The type of data in this loader
    Direct Known Subclasses:
    CacheLoader

    public abstract class BaseLoader<C,R,E,D>
extends Loader<BaseResponse<R,E,D>>
    A class that performs asynchronous loading of data. After loading completion, the registered callback will be called. See setCallback().
