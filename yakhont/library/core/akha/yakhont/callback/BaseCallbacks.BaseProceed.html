Class BaseCallbacks.BaseProceed

  • java.lang.Object
    • akha.yakhont.callback.BaseCallbacks.BaseProceed
  • Direct Known Subclasses:
    BaseCallbacks.BaseLifecycleProceed

    BaseCallbacks<T>

    public abstract static class BaseCallbacks.BaseProceed
extends Object
    The BaseProceed is intended for support of BaseCallbacks instances registration. Most implementations should not use it directly, but instead call BaseActivityLifecycleProceed.register or BaseFragmentLifecycleProceed.register.
